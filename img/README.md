Repositorio Proyecto Final del Study Jam Android Development for Beginners II 2016, llevado a cabo en la ciudad de La Paz, Bolivia a cargo del <a target="_blank" href="http://www.gdg.androidbolivia.com">GDG Android Bolivia</a> .

Nombre Aplicación.
---
El nombre de la aplicación es: Cambia Santa

Objetivo
---
Mostrar diferentes imagenes de Santa Claus

Caracteristicas
---
* Interacción con botones


Wireframes
---
Puedes visualizar los Wireframes de este proyecto.

https://ninjamock.com/Designer/Workplace/4872493

Compatibilidad
---
Esta aplicación es compatible con versiones de Android 4.2 o superior.

Uso
---------
Para probar este ejemplo clona este repositorio de la siguiente forma:
>
>     $ git clone https://github.com/sefamol/ProyectoFinal_StudyJam.git

Luego de ello dentro de Android Studio:

* File --> New --> Import Project
* Seleccionas la ruta donde hiciste el `clone` del proyecto.
* Build --> Rebuild Project
* Run

Descripción técnica
---
En este proyecto de utilizáron los siguientes componentes tanto en el `diseño` como en la `funcionalidad`:

**`Vista:`**
* LinearLayout (Horizontal) para la alineación de las vistas principales.
* RelativeLayout, para el acomodamiento de la segunda pantalla: SegundaActivity.
* Button, para cambiar en la aplicación.
* EditTexts, para recabar información acerca de los botones.
